Provably Random Raffle Smart Contract
=====================================

Project Overview
----------------

This project is a decentralized raffle (lottery) smart contract developed as part of a learning journey using Foundry. It demonstrates how to build a fair, transparent, and automated lottery system by integrating Chainlink services.

Objectives
----------

The main objectives of this project are:

1.  Allow users to enter the raffle by paying a ticket fee.
2.  Accumulate all ticket fees into a prize pool for the winner.
3.  Automatically select a winner after a predefined time interval.
4.  Ensure randomness using Chainlink VRF.
5.  Automate the raffle process using Chainlink Automation.

Features
--------

-   Raffle entry through ticket purchase
-   Automatic winner selection based on time intervals
-   Verifiable randomness using Chainlink VRF
-   Fully automated execution using Chainlink Automation

Tech Stack
----------

-   Solidity
-   Foundry
-   Chainlink VRF
-   Chainlink Automation

Learning Purpose
----------------

This project is built purely for educational purposes while learning smart contract development with Foundry.
